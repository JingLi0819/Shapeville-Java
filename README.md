# Shapeville-Java-GUI 
这是一个基于 Java Swing 开发的图形化教育/交互式系统。

### 🚀 核心功能
* **多任务管理**：系统包含 Task1 到 Task6 多个独立任务模块（如图形识别、角度计算等）。
* **图形化交互**：通过 `MainWindow` 和 `CenterPanelContainer` 构建了完整的 GUI 导航框架。
* **颜色管理系统**：内置 `ColorManager`，支持界面颜色的统一调度和刷新。
* **丰富的素材库**：项目集成了大量的图形（2D/3D）和 UI 按钮资源。

### 🛠️ 技术栈
* **语言**：Java
* **框架**：Swing / AWT (用于 GUI 设计)
* **构建工具**：Maven (包含 `pom.xml`)

### 📂 项目结构说明
* `src/main/java`: 存放核心业务逻辑代码。
* `src/main/resources`: 存放图片素材和配置文件。
* `bin/`: (已通过 .gitignore 忽略) 编译产物。
